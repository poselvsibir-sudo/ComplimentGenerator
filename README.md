**Генератор комплиментов**

Небольшое приложение на C# (WinForms), которое генерирует случайные комплименты.
Проект создан для практики языка C#, работы с формами, событиями и базовой логикой приложения.

**Функционал**

Генерация случайных комплиментов

Простое графическое окно на Windows Forms

Несколько форм

Возможность создать свой собственный комплимент

Полностью рабочий мини-проект для портфолио

**Технологии**

C#

.NET

Windows Forms

**Как запустить**

Склонируйте репозиторий:

git clone <ссылка на ваш репозиторий>

Откройте проект в Visual Studio (файл .csproj или .sln).

Соберите проект: Build → Build Solution (Ctrl+Shift+B).

Запустите: Start (F5).
